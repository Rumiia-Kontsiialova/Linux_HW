2.Что будет, если создать файл или папку в ОС Linux с одинаковым названием, но с разным регистром? (например Folder и FOLDER )?

- эти папки будут считаться разными файлами или папками, потому что Linux чувствителен в регистру.

3.Назовите временную папку в  ОС Linux и Windows?

OC Linux /tmp
OC Windows C:\Windows\Temp

4.Почему версий ОС Linux существует такое большое количество?

Одна из особенностей - открытый исходный код, благодаря этому каждый может модифицировать его под свои цели.

5.Для чего нужна папка /opt, расположенная в корневом каталоге?

Для установки дополнительных программ.

6.Для чего нужна командная оболочка?

Для взаимодействия с операционной системой через командную строку. 
Командная оболочка переводит команды, написанные на понятном для человека языке, в задачи, которые может выполнить компьютер.

7.Что такое ядро ОС?

Это главный "мозг" операционной системы. 
Оно управляет всем "железом" компьютера — процессором, памятью, жестким диском. Если программа хочет что-то сделать (например, сохранить файл), она передаёт запрос ядру, а ядро уже говорит компьютеру, что делать.
